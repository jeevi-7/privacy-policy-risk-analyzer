# Privacy Policy Risk Analyzer

##  Overview

Privacy Policy Risk Analyzer is an AI-powered web application developed using Java Spring Boot.

The application allows users to upload a Privacy Policy PDF, extracts the text, analyzes it using AI, and stores the analysis in a MySQL database.

The goal of this project is to help users better understand privacy policies by identifying potential privacy risks and presenting an easy-to-read AI-generated analysis.

---

## Current Features (Version 1.0)

- Upload Privacy Policy PDF
- Extract PDF text using Apache PDFBox
- AI-powered Privacy Policy Analysis
- Store uploaded data in MySQL Database
- Display extracted text and AI analysis
- Simple user interface using Thymeleaf
